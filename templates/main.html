Weather Watcher

Part of flying in Alaska is "weather hold." The tedious time you spend sitting in front of a screen refreshing the weather to see if it's changed enough to safely go fly. A while back I put together a tool using Flask and Bootstrap and a little Javascript to refresh the weather for me. You can find it here.

Simple Inventory

I've worked at a few places that didn't have much of an inventory control process for their air freight. We mostly just put stuff in the warehouse then manually went around looking for it when it was time to fly. I decided to try my hand at a solution to that. I created a simple web app with that can manage an inventory. You can check stuff in, check stuff out, and create new customers and users. It creates labels that have a QR code that load them into a cart. It uses Python for the back end (with flask, like this site), SQLite for the database, and a little HTML and Javascript to add functionality. Check it out here.

Nearly Continuous Integration

I have been recently reading quite a bit about CI/CD processes. I'm self-educating on the basics of devops. As part of that process, I have created a pipeline that makes it easier to push code to this site. The project lives on Github, and a script runs on python anywhere that looks for updates to the git hub repo. If there are updates, the test environment test.python.io gets updated, then deploying to production just requires running a "reload.sh" script. It's not exactly what you'd see in industry (I just have a test and a production app) but for a simple page, it's "good enough."
